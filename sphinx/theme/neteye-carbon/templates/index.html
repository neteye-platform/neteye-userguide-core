Modules
Core

Modules that are shipped with the standard NetEye image build NetEye Core. NetEye Core is a set of Modules shipped with the standard NetEye image build.

It presents most commonly used functionalities offered by the platform, including monitoring, visualization, configuration, reporting, and event handling.

All other Feature Modules can be installed on demand.

Feature

Additional Feature Modules can be installed on demand to enhance your monitoring experience with NetEye.

The NetEye modular architecture supports the installation of additional Modules that extend the NetEye Core functionalities. This separation allows to customize NetEye in order to address specific customer needs.

Feature Modules are to be installed on demand.
